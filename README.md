# 🏊‍♂️ Splashboard: Ingeniería de Datos y Análisis de Natación de Élite

## 🎯 Sobre el Proyecto
**Splashboard** es una aplicación web interactiva que ingiere, procesa y visualiza datos de competiciones de Omega Timing. El objetivo es transformar archivos crudos en formato estándar Lenex (`.lef`) en un dashboard analítico que permita explorar tiempos, clasificaciones y rendimiento de los nadadores.

Para ello se ha hecho un proceso de **ETL (Extracción, Transformación y Carga)** que incluye la limpieza y normalización de datos, así como la creación de visualizaciones interactivas.
